# Як долучитися до українського перекладу EU5

Дякуємо за інтерес до проєкту! Цей документ пояснює, як ви можете допомогти з перекладом Europa Universalis V українською мовою.

## Способи долучитися

### 1. Через GUI редактор (рекомендовано)

У репозиторії є графічний редактор для перекладу -- `tools/localization_gui.py`. Він не потребує нічого, крім Python (tkinter вбудований).

#### Початок роботи

1. **Fork** репозиторію (кнопка "Fork" у правому верхньому куті)

2. **Клонуйте** ваш fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/eu5-ukrainian-localization.git
   cd eu5-ukrainian-localization
   ```

3. **Запустіть GUI:**
   ```bash
   python tools/localization_gui.py
   ```

4. **Вкажіть папки:**
   - **Папка мода** -- автоматично визначиться (файли в `main_menu/localization/...`)
   - **Папка гри** -- вкажіть шлях до оригінальної локалізації EU5 (щоб бачити англійський текст для порівняння)

5. **Перекладайте!** GUI покаже неперекладені рядки, оригінал з гри та поле для введення перекладу.

#### Гарячі клавіші

| Клавіша | Дія |
|---------|-----|
| `Ctrl+S` | Зберегти переклад |
| `Ctrl+Enter` | Зберегти і перейти до наступного рядка |
| `Ctrl+N` / `Ctrl+P` | Наступний / попередній рядок |
| `Ctrl+F` | Фокус на поле пошуку |
| `F5` | Пересканувати файли |
| `Escape` | Скасувати зміни |

#### Commit через GUI

Коли закінчили блок роботи, натисніть **"Git Commit..."** прямо в GUI -- він покаже змінені файли та дозволить ввести commit message.

Потім:
```bash
git push origin main
```
І створіть Pull Request на GitHub.

### 2. Ручне редагування файлів

Якщо вам зручніше працювати з файлами напряму (VS Code, Notepad++ тощо):

1. Створіть гілку:
   ```bash
   git checkout -b translate/category-name
   ```

2. Відкрийте потрібний `.yml` файл у `main_menu/localization/dlc/english/`

3. Знайдіть рядки без кирилиці -- це неперекладені рядки

4. Перекладіть та збережіть (UTF-8 with BOM)

### 3. Через Issues (без Git)

Якщо ви не хочете працювати з Git:

1. Відкрийте нову Issue на GitHub
2. Вкажіть файл, номер рядка, поточний текст та ваш переклад
3. Ми застосуємо зміни

## Правила перекладу

### Формат файлів

- Кодування: **UTF-8 with BOM**
- Формат рядка: `KEY_NAME:0 "Перекладений текст"`
- **Не змінюйте** ключі (текст до двокрапки)

### Збереження тегів

Теги -- це спеціальні конструкції, які гра замінює на дані. Їх **обов'язково** треба зберігати:

```yaml
# Правильно:
"Отримати $GOLD$ золота від [Country.GetName]"

# Неправильно (теги видалені):
"Отримати золота від країни"
```

Типи тегів:
- `$VAR$` -- змінні
- `[GetName]`, `[ROOT.GetCountry]` -- скриптові виклики
- `#T`, `#!` -- форматування тексту
- `@icon!` -- іконки
- `\n` -- перенос рядка

GUI підсвічує теги синім і попереджає, якщо ви випадково видалили тег з перекладу.

### Термінологія EU5

| Англійська | Українська |
|------------|------------|
| Estate | Стан (соціальний) |
| Subject | Васал / Підлегла держава |
| Overextension | Перенапруження |
| Core | Ядро |
| Claim | Претензія |

### Уникайте москалізмів

- "любий" -> "будь-який"
- "питання" -> "пита́ння" (наголос на А)
- Тощо -- перевіряйте себе

## Про файли з `_ru_` в назві

Файли `customizable_localization_ru_*.yml` -- це **НЕ** російська мова. `RU` -- це технічний ідентифікатор Paradox для системи відмінків слов'янських мов:

| Суфікс | Відмінок |
|--------|----------|
| `_NOM` | Називний (хто? що?) |
| `_GEN` | Родовий (кого? чого?) |
| `_DAT` | Давальний (кому? чому?) |
| `_plural` | Множина (1 рік, 2 роки, 5 років) |

Назви файлів змінювати **не потрібно** -- гра шукає їх саме так. Вміст має бути українською.

## Категорії файлів

Для уникнення конфліктів, оберіть одну категорію:

| Категорія | Папка | Опис |
|-----------|-------|------|
| Events DHE | `events/DHE/` | Динамічні історичні події |
| Events Character | `events/character/` | Події персонажів |
| Events Culture | `events/culture/` | Культурні події |
| Interfaces | кореневий рівень | Елементи інтерфейсу |
| Missions | `missions/` | Місії країн |
| Government | кореневий рівень | Урядові системи |
| Modifiers | кореневий рівень | Модифікатори |

**Перед початком:** перевірте Issues, чи категорія не зайнята іншим перекладачем.
