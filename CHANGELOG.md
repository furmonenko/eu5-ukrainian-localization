# Історія змін

## Оновлення 1.0.3 (22 лютого 2026)

### Виправлення москалізмів

- Виправлено 250+ залишків російського тексту у modifier_types (назви ресурсів, парламенти, модифікатори)
- Видалено дублікат рядка з російським текстом у дипломатії
- Виправлено російські закінчення -ии на українські -ії (Гамбії, Голландії, Фінікії)
- Замінено «числа рекрутів» на «кількості рекрутів» в ігрових концепціях та інтерфейсі
- Замінено «ИИ» на «ШІ» (штучний інтелект)

### Уніфікація термінології: Русь замість Рутенії

- Замінено латинізоване «Рутенія» на історично автентичне **«Русь»** (країна, регіон, всі відмінки)
- Прикметник «Рутенський» → **«Руський»** по всій грі
- Культурна група «Рутенська» → **«Руська»**, мова «Рутенська» → **«Руська»**
- Культуру карпатських русинів перейменовано з «Русинська» на **«Карпато-русинська»**
- Оновлено всі події Литви, Києва, Волині, Туреччини, Австрії та тексти історій країн

### Переклад пропущеного тексту

- Перекладено весь файл досягнень (achievements) — 14 рядків, що залишалися англійською
- Виправлено неперекладений рядок у туторіалі: «Hints 2b» → «Поради»

### Виправлення плейсхолдерів

- Виправлено 7 самопосилальних плейсхолдерів у системі рангів (Імперія, Королівство, Герцогство, Графство, Село, Містечко, Місто)
- Додано відсутні `:0` форматери

## Оновлення 1.0.2 (26 грудня 2024)

### Виправлені баги

**Тултіп армій та флотів**
- Раніше при наведенні на армію показувало "Ця армія належить державі" без назви країни
- Тепер коректно відображається назва країни-власника

**Плашка перемир'я**
- Виправлено відображення жовтої плашки перемир'я у верхній частині екрану
- Раніше текст не відображався через проблему з апострофом у слові "перемир'я"

**Прикметники країн**
- Виправлено 2409 обрубаних прикметників країн
- Було: "Лівонськ", "Тевтонськ", "Шведськ", "Польськ"
- Стало: "Лівонський", "Тевтонський", "Шведський", "Польський"

**Назви культур та мов**
- Виправлено понад 2000 обрубаних назв культур та мов
- Було: "Польськ", "Чеськ", "Латинськ"
- Стало: "Польська", "Чеська", "Латинська"
- Додатково виправлено: Турецька, Грецька, Німецька, Мазовецька, Половецька, Ненецька та інші
- Виправлено "Кастільська" на правильне "Кастильська"

**Злиплі слова у назвах держав**
- Виправлено проблему, коли назви держав відображалися без пробілу
- Було: "ВеликеКнязівство", "МарінідськийСултанат", "ВільнеМісто"
- Стало: "Велике Князівство", "Марінідський Султанат", "Вільне Місто"

**Кнопка "Продовжити як"**
- Виправлено злиплі слова при наведенні
- Було: "Продовжити якКастилія"
- Стало: "Продовжити як Кастилія"

**Назви полків**
- Виправлено порядкові числівники для всіх культур
- Було: "1ᵉʳ-й полк", "2º-й полк" (змішані французькі/іспанські та українські суфікси)
- Стало: "1-й полк", "2-й полк"

**Ідейні групи уряду**
- "Духовний" → "Духовність"
- "Гуманістичний" → "Гуманістичність"
- "Духовний проти Гуманістичного" → "Духовність проти Гуманістичності"

**Російський текст**
- Знайдено та виправлено залишки російського тексту:
  - "исследователей" → "дослідників"
  - "Мощь" → "Міць" (артилерія, піхота, кавалерія, флот)
  - "Лояльность к сюзерену" → "Лояльність до сюзерена"
  - "Стремление к реформации" → "Прагнення до реформації"
  - "Создать новий загін" → "Створити новий загін"
  - "Голубая армія" → "Блакитна армія"
  - "Флорентийское гражданское ополчення" → "Флорентійське громадянське ополчення"
  - "Защита флангов" → "Захист флангів"
  - "Нехватка припасов" → "Нестача припасів"
  - "Подтвердите расформирование" → "Підтвердіть розформування"
  - "Скомандовать всеобщее отступление" → "Скомандувати загальний відступ"
  - та багато інших

**Англійський текст**
- Видалено "The " з повідомлень про приєднання до фракцій
- Перекладено повідомлення про облоги, дослідження та ембарго

- "Знижчення" → "Зниження"
- "вихоувані" → "виховані"
- "районух" → "районах"
- та інші

**Москалізми**
- "возмездіє" → "відплату"
- "осведомителів" → "інформаторів"
- "сброд" → "наволоч"
- "думку о" → "думку про"
- "икона" → "ікона"
